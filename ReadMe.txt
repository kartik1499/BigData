To download data using New York Times API:

Run the main method of ArticleSearchQueryImpl class. We have hardcoded 3 values: companyName, begin date and end date; which are to be passed to the query object. You can change those (or use same) and generate the articles.

The current values are:
company Name: Walmart
begin date: 01/01/2015
end date: 10/31/2015
The output will be in a folder: C:\BigDataProject.
